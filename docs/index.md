# Ars Magica Rulebook

Welcome to the complete, searchable reference for Ars Magica!

## About This Rulebook

This wiki contains the complete rules for the Ars Magica tabletop roleplaying game, 
organized for easy searching and cross-referencing.

## Quick Navigation

### Core Rules
- **Characters** - Character creation, abilities, and advancement
- **Magic** - The Hermetic magic system
- **Spells** - Complete spell listings
- **Combat** - Combat rules and mechanics

### Advanced Rules
- **Laboratory** - Laboratory activities and research
- **Saga** - Running long-term campaigns
- **Covenants** - Creating and managing covenants

### Setting & Creatures
- **Mythic Europe** - The game's historical setting
- **Realms** - The four realms of power
- **Bestiary** - Creatures and adversaries

### Reference
- **Appendices** - Glossaries and conversion notes

## Using This Wiki

- Use the **search bar** to find specific rules or topics
- Navigate by **chapter** using the sidebar
- Cross-references are linked throughout
